# Systems-monitor
## For my systems project

Installer:
```
   chmod +x installer.bsh
```
To run:
```
   ./setup_face_tracker.sh
```
Verify:
```
    chmod +x verify_tracker_install.sh
```
To run:
```
   ./verify_tracker_install.sh
```
## Start program:

#### after install, and if you have already installed

    ~/tracker-file/start_tracker.sh
